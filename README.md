# Sistema de Monitoramento Free Flow para Pedágios
Este projeto foi desenvolvido como parte de uma iniciativa educacional na Faculdade FIAP, com foco na criação de uma solução inovadora para o monitoramento de pedágios que operam com o sistema Free Flow. O objetivo principal é oferecer uma ferramenta eficaz que permita o acompanhamento em tempo real das atividades nos pedágios, otimizando tanto a experiência do usuário quanto a eficiência operacional.

## 👀 Visão Geral
![image](https://github.com/BarbaraMAlmeida/Challenge_FIAP_ADS/assets/84252525/b7046cb5-9309-4524-81d1-f59aa1dd783d)
Nosso sistema trás novas funcionalidades que vão além do gerenciamento das câmeras e sensores de um pórtico free flow.
Controle as câmeras do pedágio por vídeos em tempo real e receba status das suas funcionalidades.
Você também pode ter a previsão do tempo na palma da mão, recebendo alertas sobre problemas que possam afetar seus equipamentos. 
Abra chamados para equipamentos com problemas e expanda a utilização do sistema para outros setores da empresa.

## 🛠️ Recursos Principais
Nosso sistema funciona com 3 tipos de acesso: 

Administrador -> Acesso para quem pode gerenciar qualquer pórtico. Esse perfil tem poder de escolher qual pórtico acompanhar, ter acesso à status e vídeos das câmeras, abrir chamados para falha de equipamentos, ver os pagamentos em tempo real e realizar cadastro de login para outros setores e perfis.

Central -> Acesso para funcionários que trabalham na central de controle de um pórtico específico. Ele poderá gerenciar câmeras, status, pagamentos e abrir chamados somente do pórtico que está vinculado.

Suporte -> Acesso para funcionários do suporte técnico, responsáveis pela comunicação com o cliente. Eles terão acesso aos pagamentos e seus detalhamentos, assim, podendo auxiliar os usuários que estiverem com alguma dúvida e contatarem o suporte.
